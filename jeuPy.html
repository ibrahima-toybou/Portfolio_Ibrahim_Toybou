Création d’un Jeu d'Aventure en Python

Retour

Contexte

Ce projet a été réalisé dans un cadre pédagogique visant à mettre en pratique les bases de la programmation en Python.

L’objectif était de concevoir un jeu d’aventure textuel interactif, entièrement joué dans le terminal. Ce type de jeu met en œuvre des structures de contrôle, des fonctions, des listes et des conditions.

Objectifs

  1. Créer un jeu d’aventure simple et interactif
  2. Faire évoluer le joueur dans des forêts sombres et prendre des décisions
  3. Utiliser les bases de Python : fonctions, boucles, conditions
  4. Afficher une narration fluide dans le terminal
  5. Rendre le jeu amusant

Le fonctionnement du jeu

Le joueur incarne un aventurier qui explore forêts mystérieuse. À chaque étape, il doit choisir une action : se déplacez vers la gauche ou vers la droite.

Chaque choix a des conséquences : certains mènent à la victoire, d’autres à la rencontre d'un démon aux yeux rouge qui lui fait perdre des 10 poins. Le jeu est entièrement basé sur du texte et se joue via l’invite de commandes.

Exemples de mécanique


  • Utilisation de input() pour demander des choix
  • Conditions if / elif / else pour l’arbre de décisions
  • Variables pour stocker l’état du joueur (inventaire, vie, etc.)
  • Affichage dynamique via print()

  • Technologies utilisées :
    • Python
    • Invite de commandes (Terminal)

  • Fonctionnalités du jeu :

    • Exploration de lieux avec narration textuelle
    • Choix multiples influençant la suite de l’aventure
    • Gestion d’un petit inventaire
    • Fin de partie : victoire ou défaite
