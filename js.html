JavaScript - Langage de programmation pour le web

Retour

JavaScript est un langage de programmation essentiel pour ajouter de l'interactivité et des fonctionnalités dynamiques aux pages web. Contrairement à HTML (structure) et CSS (style), JavaScript permet de manipuler les éléments d'une page, de répondre aux actions de l'utilisateur et de communiquer avec des serveurs pour charger ou envoyer des données sans recharger la page.

  1. Je maîtrise la manipulation du DOM (Document Object Model) avec des méthodes comme document.querySelector et innerHTML.
  2. Je maîtrise la gestion des événements comme les clics, survols, et saisies avec des écouteurs tels que addEventListener.
  3. Je maîtrise l'utilisation de fetch pour effectuer des requêtes API simples (par exemple, charger des données depuis un serveur).
  4. Je maîtrise le débogage avec la console JavaScript du navigateur.
  5. Je suis encore en apprentissage pour la programmation asynchrone avancée avec async/await et la gestion des promesses complexes.
  6. Je suis encore en apprentissage pour l'utilisation avancée des frameworks comme React ou Vue.js.
