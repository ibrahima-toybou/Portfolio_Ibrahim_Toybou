CSS - Feuilles de style en cascade

CSS (Cascading Style Sheets) est un langage de style utilisé pour décrire la présentation d'un document HTML. Il permet de contrôler la mise en page, les couleurs, les polices, les marges et bien d'autres aspects visuels d'un site web. Grâce à CSS, un site peut être transformé d'un contenu brut à une interface utilisateur esthétique et fonctionnelle.

  1. Je maîtrise la création de styles de base pour les éléments HTML (couleurs, polices, marges, alignements).
  2. Je maîtrise l'utilisation de sélecteurs simples et avancés (classes, identifiants, attributs).
  3. Je maîtrise la gestion des mises en page avec Flexbox et Grid.
  4. Je maîtrise l'intégration de médias responsifs pour l'adaptation des sites à différents écrans.
  5. Je suis encore en apprentissage pour Les préprocesseurs CSS comme Sass ou Less.
