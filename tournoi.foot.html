Contexte

Dans un cadre pédagogique, j'ai réalisé un projet de base de données portant sur l’organisation d’un tournoi de football. L’objectif était de modéliser et gérer les données liées aux équipes, joueurs, matchs et résultats.

Le projet a été mené en plusieurs étapes : de la modélisation conceptuelle jusqu’à la création et exploitation de la base avec SQL.

Objectifs

  1. Concevoir un Modèle Conceptuel de Données (MCD)
    2. voir un extrait
  2. Créer un Modèle Logique de Données (MLD)
    3. voir un extrait
  3. Générer la base de données en SQL
    4. voir un extrait
  4. Permettre la gestion des équipes, joueurs et matchs

Étapes du projet


  • Création du MCD avec les entités : Équipe, Joueur, Match, Résultat
  • Traduction en MLD, puis en script SQL
  • Création de la base dans MySQL
  • Insertion de données et tests de requêtes
  • Technologies utilisées :
    • SQL
    • MySQL
    • Looping pour le MCD
  • Fonctionnalités de la base :
    • Ajout d’équipes et de joueurs
    • Programmation et enregistrement des matchs
    • Stockage des scores et résultats
    • Requêtes de classement par points
