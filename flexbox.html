Contexte

Ce projet a été réalisé dans le cadre d’un projet personnel.

J’ai développé une plateforme de gestion de budget en Java, permettant aux utilisateurs de suivre leurs dépenses, revenus et soldes de manière centralisée et sécurisée.

Objectifs du projet

  1. Mettre en pratique mes compétences en Java
  2. Mettre en place une base de données relationnelle (MySQL)
  3. Créer une interface utilisateur fonctionnelle
  4. Gérer les opérations CRUD sur les données financières
  5. Structurer un projet complet de bout en bout

Fonctionnalités de la plateforme

L'application permet aux utilisateurs d’ajouter, modifier, supprimer et consulter des opérations financières. Elle offre une vue d’ensemble claire du solde actuel, des revenus et des dépenses.

L’ensemble des données est stocké dans une base de données MySQL, et les interactions sont gérées via JDBC.
